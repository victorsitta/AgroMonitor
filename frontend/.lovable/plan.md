

# AgroMonitor IoT — Plataforma de Monitoramento Agrícola

## Página 1: Landing Page
- Hero section com animação de partículas/dados fluindo, título impactante e CTA "Comece Agora"
- Seção "Como Funciona" com 3 steps animados (Conecte Sensores → Monitore em Tempo Real → Tome Decisões)
- Seção de features com cards interativos (Temperatura, Umidade, Qualidade do Grão, Alertas)
- Seção de grãos monitorados com ícones animados (Feijão, Trigo, Milho, Soja, Arroz, Café)
- Seção de estatísticas animadas (contadores: sensores ativos, fazendas, toneladas monitoradas)
- Seção de depoimentos/clientes com carrossel
- Seção de planos/pricing
- Footer com links e informações de contato
- Navbar fixa com efeito de scroll (transparente → sólida)

## Página 2: Login
- Formulário de login com validação nos campos (email e senha)
- Design split-screen: lado esquerdo com imagem/ilustração agrícola, lado direito com formulário
- Link para registro e "Esqueci minha senha"

## Página 3: Registro
- Formulário com campos: nome, email, telefone, nome da fazenda, senha
- Validação em tempo real nos campos
- Mesmo layout split-screen do login

## Página 4: Dashboard Principal (após login)
- Sidebar com navegação (Dashboard, Sensores, Grãos, Alertas, Relatórios, Configurações)
- Cards de resumo no topo: Total de Sensores, Alertas Ativos, Temperatura Média, Umidade Média
- Gráficos interativos com Recharts: temperatura e umidade ao longo do tempo (dados mock)
- Mapa/grid dos silos com status visual (verde/amarelo/vermelho)
- Lista de alertas recentes

## Página 5: Monitoramento de Grãos
- Tabs para cada grão (Feijão, Trigo, Milho, Soja, Arroz, Café)
- Para cada grão: gráficos de temperatura, umidade, qualidade
- Tabela com dados dos sensores associados
- Indicadores de status com cores

## Página 6: Sensores
- Lista/grid de sensores com status (ativo/inativo/alerta)
- Cards com informações: localização, último dado recebido, bateria
- Filtros por tipo e status

## Página 7: Alertas
- Lista de alertas com severidade (crítico, aviso, info)
- Filtros por tipo e status
- Detalhes do alerta ao clicar

## Design & Interatividade
- Tema verde/escuro profissional remetendo ao agronegócio
- Animações suaves nas transições de página e hover nos cards
- Contadores animados na landing page
- Gráficos interativos com tooltips no dashboard
- Dados mock realistas para demonstração
- Design totalmente responsivo
- Navegação funcional entre todas as páginas
